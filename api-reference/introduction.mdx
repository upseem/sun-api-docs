## 通用约定

### 请求头

所有接口都需要 `Content-Type: application/json` 和对应格式的鉴权头。详见[鉴权](/authentication)。

### 错误响应

错误统一返回如下结构：

```json
{
  "error": {
    "message": "错误描述",
    "type": "invalid_request_error",
    "code": "model_not_found"
  }
}
```

### 流式响应

支持流式的接口在请求体中传 `"stream": true` 即可，响应为 SSE（Server-Sent Events）格式。
