## 它是什么

SUNX AI API 是一个 AI API 中转网关，把各家 AI 提供商（OpenAI、Anthropic、Google、AWS Bedrock、Azure、阿里通义、字节豆包、Replicate、Midjourney、Suno 等 40+）聚合到统一接口下，并完整兼容三种主流 API 格式。

<CardGroup cols={3}>
  <Card title="OpenAI 格式" icon="bolt">
    `POST /v1/chat/completions`
    `POST /v1/responses`
    `POST /v1/images/generations`
  </Card>
  <Card title="Anthropic 格式" icon="message">
    `POST /v1/messages`
    完全兼容 Claude 官方 SDK
  </Card>
  <Card title="Gemini 格式" icon="sparkles">
    `POST /v1beta/models/*:generateContent`
    完全兼容 Google GenAI SDK
  </Card>
</CardGroup>

## 核心特性

<CardGroup cols={2}>
  <Card title="协议自动互转" icon="arrows-rotate">
    用 OpenAI SDK 调 Claude 模型，或者用 Claude SDK 调 GPT —— 都自动适配，无需改代码。
  </Card>
  <Card title="任意上游" icon="globe">
    OpenAI、Claude、Gemini、Bedrock、各国产模型、自部署 Ollama / vLLM 全支持。

    Midjourney、Sora、Kling、Vidu、Jimeng、Hailuo、Replicate、Suno 等异步任务接口全适配。

  </Card>
</CardGroup>
